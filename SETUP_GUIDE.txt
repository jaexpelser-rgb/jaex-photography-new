# Jaex Photography - Deployment Guide

## FREE Hosting on Render.com (Recommended)

### Step 1: Create a free GitHub account
1. Go to https://github.com
2. Sign up for free
3. Create a new repository called "jaex-photography"
4. Upload all project files (except node_modules)

### Step 2: Deploy on Render
1. Go to https://render.com
2. Sign up with your GitHub account (free)
3. Click "New +" → "Web Service"
4. Connect your jaex-photography repository
5. Settings:
   - Name: jaex-photography
   - Build Command: npm install && npm run build
   - Start Command: npm start
6. Click "Create Web Service"
7. Wait 2-3 minutes for it to deploy
8. Your site is live at: https://jaex-photography.onrender.com

### Step 3: First-Time Setup
1. Visit https://jaex-photography.onrender.com/setup
2. Create your admin account
3. Log in and start adding photos!

## Upload Your Logo
Put your logo.png file in the public/ folder before uploading to GitHub.
